# Website

网站基于 [Docusaurus 2](https://docusaurus.io/)改进开发。计划支持的功能有：

- WebOffice的嵌入
- 个性化的主题
- 后期计划：将所有的笔记的raw文件和web仓库文件全部分离，实现统一样式和自动化发布
